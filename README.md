


# FreshMart – Smart Retail Platform

FreshMart is a GenAI-powered retail application designed to increase
Average Basket Size (ABS) through real-time personalization and
context-aware product recommendations.

## Problem Statement
FreshMart operates both online and offline stores. Customers often
purchase fewer items per transaction due to forgotten needs, generic
recommendations, and lack of personalization, leading to a lower
Average Basket Size.

## Objective
Increase the Average Basket Size by intelligently suggesting relevant
add-on products based on:
- Customer purchase history
- Frequently bought items
- Wishlist and saved items
- Current cart context

## Solution Overview
FreshMart uses a combination of backend logic and Generative AI to:
- Identify items a customer is likely to buy but missed
- Avoid recommending items already present in the cart
- Generate natural-language explanations for recommendations
- Support both online shopping and in-store POS recommendations

## Tech Stack
- Backend: Node.js, Express
- Database: MongoDB
- Frontend: React (planned)
- AI: Generative AI for reasoning and personalization
- Version Control: Git & GitHub
